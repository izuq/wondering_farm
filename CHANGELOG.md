# 更新日志

## 2026-05-26

### 重构
- **移除 PIL 依赖**：删除 Pillow 图片加载/缩放缓存，作物渲染全部改用纯 tkinter canvas 绘图函数。项目现在零外部依赖，完全基于 Python 标准库运行
- **清理废弃文件**：删除 `archive_v2.1/`（旧版 QML/重构备份）、`pictures/`（作物图片）、`requirements.txt`

### 修复
- **农业建筑名称不显示**：加工厂/酿酒厂的顶部名称文字在已建造地块上不显示，补全 `_update_agro_grid()` 中缺失的 `create_text` 调用

### 新增
- **双地块页面**：地块1/地块2 分页切换，各 50 块独立土地，独立的解锁进度
- **饲料商店**：商店新增饲料标签页，四种饲料可直接购买

### 改进
- **金色南瓜**：生成概率从 1% 提升到 2%，免疫所有负面事件（虫灾、暴风、外星人攻击）
- **虫灾效果调整**：从产量减半改为延长生长 2 小时
- **UI 优化**：地块背景色改为浅色系（仿养殖场），剩余时间从左上移到地块底部居中，农业建筑底部显示时间而非加工物名称，字号统一

## 2026-05-22

### 新增
- **出售动物功能**：点击养殖场栏位中的动物，弹出操作对话框，新增"出售动物"选项。出售价格基于购买价 50% + 栏位等级加成 + 产出次数加成 + 动物折扣天赋加成，出售前自动收集待收产品
- **金色南瓜彩蛋**：南瓜成熟时有 1% 概率变成金色南瓜，需要再长一个完整生长周期才能收获，价值为普通南瓜 10 倍（12000💰）。金色南瓜不会被一键收获，只能手动点击土地格子单独收获

### 改进
- **土地/动物弹窗升级**：土地和养殖场网格显示等级（Lv.X），侧边栏移除升级按钮，改为点击格子弹窗操作。升级不关闭弹窗，可连续升级，信息实时刷新

### 重构
- **配置外置**：动物、工厂、饲料、天赋、事件、成就数据从代码硬编码迁移到 `data/*.json`，新增 `reload_config()` 热加载，加新动物/工厂只需改 JSON
- **事件注册系统**：事件效果改为注册表模式，`register_event_handler()` 注册，`get_event_handler()` 分发，新增事件无需改主流程

## 2026-05-21

### 新增
- **外星人飞船随机事件**：新增 `alien_attack` 随机事件，外星人飞船随机降临摧毁 1~3 块作物，同时补偿玩家 1000💰
- **Canvas 农作物/动物图标**：土地网格和养殖场网格从纯文字按钮升级为 Canvas 绘制，6 种作物（小麦、玉米、水稻、玫瑰、胡萝卜、南瓜）和 10 种动物（鸡、鸭、兔、鹅、羊、猪、牛、羊驼、马、鹿）均有自定义图标

### 修复
- **修复登录时崩溃问题**：修复游戏登录过程中因数据加载异常导致的程序崩溃
- **修复自动收集天赋无效**：点亮"自动收集动物产品"天赋后，tkinter 版刷新时未触发收集逻辑，现已补上
- **修复出售商品无法选择数量**：商品出售时缺少数量选择逻辑，导致无法部分出售
- **修复神秘商人折扣函数缺失**：`farm_tkinter_v2.py` 导入 `get_merchant_discount` 报错，在 `farm_game_v2.py` 中新增该函数及对应的 `merchant_visit` 随机事件
- **修复天赋树界面空白**：`farm_tkinter_v2.py` 中天赋元组解包索引错位，导致 `name` 拿到分组名而非天赋 ID，渲染异常
